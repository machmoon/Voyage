# Voyage ✈️

A study-session app where every focus session is an airline flight. Pick a destination on a realistic 3D globe, choose your seat, rip your boarding pass, and study through a live airplane-window view until you land. Leave mid-flight and the plane diverts.

- **Short-haul** = 2 hours of focus. **Long-haul** = 6 hours, always booked as a connection with a real lounge break between legs (a Pomodoro in disguise — miss the final call and the connection is gone).
- Procedurally generated cabin ambience, cabin chimes, and spoken PA announcements (welcome aboard, captain's midpoint, descent, landing).
- SwiftData logbook with passport stamps, real great-circle miles, day streaks, and Silver/Gold/Platinum tiers that unlock cosmetic upgrades (business-class cabin, sunset/aurora window scenes, premium chime).

## Requirements

- **Xcode 15 or newer** (iOS 17 SDK). Install from the Mac App Store.
- No third-party dependencies. The `Voyage.xcodeproj` in this repo is generated by [XcodeGen](https://github.com/yonaskolb/XcodeGen) from `project.yml`.

## Run it

1. Install Xcode from the Mac App Store, launch it once, and let it install the iOS platform when prompted (Settings → Components if it doesn't ask).
2. Point the command-line tools at the full Xcode: `sudo xcode-select -s /Applications/Xcode.app`
3. Open `Voyage.xcodeproj` (double-click, or `open Voyage.xcodeproj`).
4. Select the **Voyage** scheme and an iPhone simulator (or your device), then **⌘R**.
   - Running on a device needs a (free) Apple ID under Signing & Capabilities → Team.
5. If you edit `project.yml` or add files outside Xcode, regenerate the project with `xcodegen generate`.

## Notes

- **WeatherKit** is coded defensively: destination weather themes the approach when the entitlement is available (paid developer account + capability enabled), and silently falls back to clear skies otherwise. Nothing to configure.
- **Location** is requested once, only to pick your nearest home airport; you can override it in Settings.
- **Background audio** is enabled so the cabin ambience keeps playing during the 30-second grace period if you background the app mid-flight — overstay and the flight diverts.
- Fast testing tip: shorten sessions by editing `shortHaulDuration` / `longHaulDuration` in `Voyage/Models/RoutePlanner.swift` and `layoverDuration` for the lounge.

## Structure

```
project.yml               XcodeGen project spec (app target, Info.plist, iOS 17)
Voyage/
  App/                    @main entry + root stage router
  Models/                 Airports, route/duration engine, SwiftData logbook, tiers
  Flight/                 FlightSession — the multi-leg state machine driving everything
  Audio/                  Procedural ambience engine, PA announcer, haptics
  Support/                Settings, location, scheduler/notifications, weather, theme
  Views/
    Home/                 MapKit globe, booking, schedule sheet
    Boarding/             Seat map, check-a-bag, boarding-pass rip, flight-mode prompt
    Flight/               Window scene, in-flight screen, layover lounge, diverted
    Landing/              Welcome, baggage claim, passport stamp
    Logbook/  Settings/
```
